    Preguntas frecuentes sobre Brightcove Audience

    Este tema contiene algunas de las preguntas comunes que se hacen acerca de Brightcove Audience.

    Por qué público y cómo funciona

    ¿Por qué necesito realizar un seguimiento de vídeo en mi solución de automatización de marketing?
    Las soluciones de automatización de marketing no realizan un seguimiento de vídeo hoy en día. Si está invirtiendo en vídeo como una potente pieza de contenido, debe asegurarse de que está rastreando su eficacia y automatizando las comunicaciones con los clientes potenciales o usuarios que ven su contenido.
    ¿Brightcove y el módulo Audience utilizan cookies?
    El módulo Audience crea y utiliza una cookie de terceros para identificar a los usuarios cuando se realiza una integración. De esta manera, no se pide a los espectadores que rellenen un formulario de prospecto por segunda vez. El módulo también utiliza las API de Marketo, HubSpot y Eloqua, que a su vez utilizan cookies para rastrear a los espectadores. Al utilizar Perfiles de audiencia, se utiliza una cookie de Brightcove.
    ¿Brightcove Audience está habilitado para todos los clientes?
    No. La audiencia está disponible como un complemento comprable para Video Marketing Suite (VMS). Sin embargo, los perfiles de audiencia están disponibles para todos los clientes de Brightcove de forma gratuita. Se puede pensar en ello como «audiencia-lite».
    Quién es elegible para Perfiles de audiencia.
    A partir del cuarto trimestre de 2017, todos los clientes de VMS tendrán acceso a perfiles de audiencia sin costo alguno. Los clientes tendrán que activarlo para comenzar a rastrear los datos. Los clientes de Enterprise Video Suite (EVS) también tendrán disponible el seguimiento de SSO de audiencia.

    Seguimiento y limitaciones del visor

    ¿Qué limitaciones tienen los perfiles de audiencia en comparación con el módulo Audience con todas las funciones?
    • Sin integraciones con sistemas de terceros
    • Sin acceso a la API
    • No hay formularios de prospecto personalizados (sólo formularios de clientes potenciales de audiencia)
    • 1000 vistas/máximo de captura de 24 horas
    Aún puedes exportar manualmente los datos de Audience a un archivo CSV o XLS.
    ¿Qué limitaciones/requisitos tiene Audience cuando se conecta a una plataforma de automatización de marketing?
    Las limitaciones y requisitos se describen en la documentación de cada plataforma:
    • Oráculo Eloqua
    • HubSpot (APIdel lado del cliente o API REST)
    • MarketoMunchkin o API REST)
    • Salesforce
    ¿Hay algún límite sobre la frecuencia con la que se puede sondear la API de audiencia? Sé que hay un límite de 1000 en los resultados de búsqueda de videos, pero ¿hay otras mayúsculas relacionadas con la frecuencia con ese extremo de API?
    El número máximo de eventos de vista por llamada API es en realidad 100. No hay límite de velocidad en su lugar, pero si intentas llamarlo demasiado, probablemente se ralentizará un poco.
    ¿Hay alguna forma de obtener datos de Audience más rápido que el retraso de 3 horas que está presente actualmente? Me gustaría obtener métricas de «cerca de tiempo real».
    No. Si quieres métricas en tiempo real, necesitarás escribir tu propio plugin y enviar datos a tu propio sistema.
    ¿Cuál es el desencadenador para que una vista de evento se envíe a Audiencia? ¿Es cuando se completa el vídeo, cuando se cierra la ventana del reproductor o algo más? ¿Qué sucede si el usuario nunca completa el vídeo y no cierra la ventana del reproductor?
    Varía, pero, básicamente, el plugin Audience envía un latido de nuevo al servidor cada 10 segundos.
    ¿Cuándo considera la Audiencia que el seguimiento en una vista está «completado»? ¿Es cuando se cierra la ventana del reproductor o si completan el vídeo? ¿Qué sucede si ven parcialmente, pausan el vídeo y nunca cierran la ventana del reproductor?
    Es cuando han pasado 15 minutos desde la última vez que vimos un latido del corazón, básicamente falta de actividad.
    ¿Es posible que los datos del espectador cambien una vez que estén disponibles en los datos de visualización de audiencia? ¿Se actualizarán esos datos de eventos en el futuro mediante algún tipo de consolidación de registros?
    Una vez que los datos están disponibles en Audiencia a través de la interfaz de usuario o API, se consideran «resueltos» y no van a cambiar.

    Protección de datos y retención de registros

    ¿Es compatible con Audience GDPR?

    Sí, Audience ofrece a los clientes la posibilidad de actuar en nombre de sus usuarios, buscando y depurando cualquier dato que hayan recopilado a través de Audience.

    https://audience.support.brightcove.com/admin/managing-viewer-data.html

    ¿Cuánto tiempo pueden conservarse los datos en Audience?
    Las integraciones de MAP se pueden establecer en un máximo de 6 meses, y las integraciones que no son de MAP se pueden establecer en 3 años.
    ¿Cuánto tiempo se almacenan en caché las búsquedas GUID?
    30 días para conocidos y 7 días para desconocidos (solo se aplica a integraciones de MAP, no se necesitan búsquedas para integraciones que no sean de mapa)

    Automatización de marketing y análisis de espectadores

    ¿Qué tiene de diferente el seguimiento de vídeos desde cualquier otro medio en mi plataforma de automatización de marketing?
    A diferencia de una página web o un documento técnico descargable, con un vídeo, puede determinar exactamente cuánto del vídeo vio el espectador y, por lo tanto, exactamente a qué contenido se ha expuesto.
    ¿Cómo se conecta con mi cuenta Eloqua/Marketo/Hubspot?
    Usted puede iniciar sesión en su cuenta de Eloqua/Marketo/Hubspot directamente a través de Video Marketing Suite en el módulo Audience. A partir de ahí, podrá crear reproductores y formularios de prospectos con el seguimiento MAP integrado, para que pueda comenzar a realizar el seguimiento de análisis de vídeo inmediatamente.
    ¿Puedo realizar un seguimiento de las vistas en mis páginas de destino de Eloqua/Marketo/Hubspot, así como en mis páginas de destino no Eloqua/Marketo/Hubspot (SiteCore/WordPress/Gallery, etc.)?
    Sí, siempre y cuando el reproductor que se está utilizando esté «Audiencia habilitada» (tenga instalado el plugin Audience), todas las vistas se rastrearán tanto si el jugador reside en una página de destino de MAP como en cualquier otra página de destino.
    ¿Qué puedo hacer un seguimiento con esta integración?
    Con nuestra integración, puede realizar un seguimiento de análisis de visualización de vídeo (vídeo, ID de vídeo, porcentaje de visualización, tiempo de visualización) de contactos conocidos que ya están en su base de datos.
    ¿Qué pasa con los contactos desconocidos? ¿Puedo rastrearlos también?
    Con Brightcove Audience, puede realizar un seguimiento de contactos desconocidos y, a continuación, cuando envíen un formulario Eloqua/Hubspot (hacerse conocido), Audience puede asociar su información de audiencia con ese nuevo contacto retroactivamente. Audience reenvía información de audiencia diariamente para que coincida con los datos de vista de contactos desconocidos a la instancia de Oracle Eloqua/HubSpot.
    ¿Dónde aparecerán mis datos de la vista de vídeo en mi MAP?
    Esto depende del MAP que esté utilizando. Consulte la siguiente documentación para Oracle Eloqua, Marketo y HubSpot.
    ¿Dónde aparecerán mis datos de visualización de vídeo en Salesforce?
    Las vistas de vídeo se entregan en Salesforce como una tarea bajo el contacto del espectador en la sección Actividad. Consulte Visualización de datos de vídeo en Salesforce para obtener un ejemplo de esto.
    ¿Puedo recibir notificaciones con Salesforce cuando uno de mis contactos ha visto vídeos?
    No directamente. Sin embargo, puede ejecutar informes que le notifiquen la actividad de un contacto.
    ¿Hay alguna diferencia entre el seguimiento con Salesforce frente a Oracle Eloqua/Marketo?
    La mayor diferencia es que Salesforce no utiliza cookies para rastrear a los usuarios. Por lo tanto, se debe pasar un identificador único para identificar al espectador. Consulte este documento para obtener más información. Además, Salesforce, al ser un CRM, no ofrece seguimientos automatizados ni puntuación de clientes potenciales. Sin embargo, puede crear tareas en Salesforce, que luego se asignan a su equipo de ventas para el seguimiento de la vida real.
    Audience tiene una opción de integración «personalizada», ¿con qué otras plataformas puede integrar Audience usando esto?

    La audiencia puede «integrarse» con cualquier plataforma de datos que:

    1. Utiliza un identificador único para cada registro de contacto
    2. Tiene una API que puede recibir datos adicionales de plataformas de terceros (como Audiencia)

    Formularios de Lead

    ¿Cuándo muestra Audience un formulario de prospecto?
    La configuración predeterminada sólo muestra el formulario de prospecto a un usuario desconocido de la base de datos de MAP (los que desea convertir en un usuario conocido). Puede anular esto en la configuración si elige mostrar siempre el formulario de prospecto. Esto es más útil para probar. Se realiza un seguimiento automático de los usuarios conocidos y esos datos se asocian con el contacto del MAP sin necesidad de mostrar un formulario.
    ¿Puedo personalizar los formularios de prospecto?
    Puede ajustar el texto y los campos mostrados mediante la interfaz de usuario. No se puede modificar más allá de lo que está disponible en la interfaz de usuario (Gallery puede actualizar algunos estilos para que los formularios coincidan con el sitio)

    Temas generales

    ¿Qué código de inserción debo usar? ¿Iframe o en la página?
    Para que Audience funcione, es necesario utilizar el código de inserción avanzado. Si se utiliza el código de inserción estándar (iframe), entonces el reproductor se aísla de la página, lo que significa que el plugin Audience no puede ver el identificador único y, por lo tanto, todos los usuarios se verán como «desconocidos».
    ¿Qué es la automatización de marketing?
    La automatización de marketing es una tecnología que rastrea la interacción de los conductos/usuarios con diferentes canales de marketing online (correo electrónico, web, formularios, contenido, vídeo, redes sociales, etc.). La automatización de marketing ayuda a los comercializadores de marketing de manera más eficaz a través de estos canales y automatiza las tareas de marketing repetitivas (como las comunicaciones por correo electrónico). La automatización de marketing permite al vendedor configurar tareas automatizadas y personalizadas basadas en variables predefinidas. Por ejemplo, si un usuario ve el 75% de un vídeo específico, envíe un correo electrónico sobre ese tema específico, pero si solo ve el 12%, envíale algo más.
    ¿Brightcove es una empresa de automatización de marketing?
    No, Brightcove ofrece una solución completa para el marketing de vídeo. Con nuestra solución, puede obtener sus vídeos en línea, organizar su contenido, crear reproductores, crear portales de vídeo, realizar un seguimiento de análisis e integrarse con su propia plataforma de automatización de marketing existente. Brightcove integra el análisis de vídeo en plataformas de automatización de marketing.
    ¿Cómo puedo comprobar si una actividad personalizada de Marketo está configurada correctamente?
    Audience tiene una URL oculta que ofrece algunas herramientas de depuración para ciertos tipos de integraciones, siendo Marketo REST una. https://studio.brightcove.com/products/videocloud/audience/debug
    ¿Qué hay que hacer para configurar una integración personalizada?
    Hay que hacer tres cosas:
    1. Debe exponer un identificador único en la URL o código de la página que Audience puede capturar.
    2. Cree un script que utilice las API de Audience para recuperar los datos como un archivo JSON (Brightcove Global Services puede ayudar con esto).
    3. Asigne los datos a los campos de su base de datos de destino (piense en la combinación de hojas de cálculo).
    La siguiente documentación de soporte cubre estas tareas:
    • Creación de integraciones personalizadas con el módulo Audience - Cubre cómo proporcionar un identificador único que Audience puede capturar
    • Audience API - Cubre cómo recuperar mediante programación todos los datos como un archivo JSON que luego puede importar en su sistema de procesamiento de datos de elección
    ¿Cómo empuja Audience los datos de CDO a Eloqua?
    Utilizamos la API masiva de objetos personalizados. https://docs.oracle.com/cloud/latest/marketingcs_gs/OMCAC/api-bulk-2.0-customobjects.html
    ¿Cómo impulsa Audience la actividad de activos externos? ¿Cómo puedo controlar la campaña, el activo externo y los tipos de actividad utilizados?
    Utilizamos la API de actividad externa (https://docs.oracle.com/cloud/latest/marketingcs_gs/OMCAC/op-api-rest-2.0-data-activity-post.html). No hay control sobre el nombre de la campaña, el nombre del activo o el tipo. Simplemente creamos una campaña por nuestra cuenta (llamada BrightCoveVideoPlay) y luego procedemos a crear actividades, también llamadas BrightCoveVideoPlay.
    ¿Qué direcciones IP puedo incluir en la lista blanca para que Audience pueda hablar con el servicio X?
    Audience ha establecido un conjunto de direcciones IP que deben utilizarse para la creación de listas blancas de IP. Para ver la lista, edite una conexión de Eloqua y haga clic en el botón ¿ No se pueden ver los datos de audiencia en Eloqua? para ver la lista de direcciones IP.
