    Creación de integraciones personalizadas con el módulo Audience

    Este tema explica cómo configurar el módulo Audience para una integración personalizada.

    El módulo Audience permite que Video Cloud sincronice los datos de visualización de los reproductores Brightcove habilitados para audiencias con plataformas populares de automatización de marketing. Además, el módulo Audience puede configurarse con una integración personalizada. De este modo, los datos de visionados se guardan localmente en Video Cloud y la Audience API se emplea para extraer los datos mediante programación.

    La integración del módulo Audience consta de dos componentes principales:

    • Un complemento del reproductor de Brightcove que permite realizar un seguimiento de los datos de visionados de los vídeos
    • Un desarrollo personalizado que emplea la Audience API para recuperar los datos de visionados

    Requisitos

    Requisitos para poder utilizar el módulo Audience:

    • Disponer de una cuenta Professional o Enterprise de Video Cloud
    • Contar con un módulo Audience activado en la cuenta, para lo cual se debe enviar la información solicitada en la pantalla de bienvenida inicial que aparece al abrir el módulo Audience.
    • Revisar y aceptar el Contrato de Servicios la primera vez que acceda al módulo.
    • Para conocer el número de clientes potenciales, se debe configurar un formulario de captación de clientes potenciales, que deberá asignarse a un reproductor
    • Utilice el código de inserción avanzado (en la página) al publicar su reproductor; el código de inserción estándar (iframe) no funcionará ya que el complemento se ejecutará dentro de un iframe y, por lo tanto, no podrá acceder a los parámetros de URL de la página principal

    Configurar la conexión

    Para configurar una integración personalizada, siga estos pasos.

    1. Abra el módulo Audience y haga clic en Conexiones de datos en la navegación izquierda.
    2. Haga clic en Añadir nueva conexión.
    3. Seleccione la integración Personalizada.
      Nota: Las integraciones disponibles se basan en su suscripción a Video Cloud.
    4. (Opcional) Escriba un nombre de conexión.
    5. Haga clic en Conectar.

    La conexión personalizada está configurada y lista para usar.

    Configurar la integración

    Una vez realizada la conexión, podrá configurar la integración.

    1. De forma predeterminada, el estado de conexión será Habilitado. El estado de conexión determina si se guardan los datos de interacción de vídeo de Video Cloud.
      • Habilitado : los datos se recopilan y se guardarán
      • Desactivado : no se recopilan datos (la integración está desactivada)
    2. Seleccione una política de conservación de datos. La política de conservación de datos determina cuánto tiempo se conservan los datos de visionados y de los formularios de captación de clientes potenciales (en caso de utilizarlos) en su cuenta de Video Cloud.
      Nota: Si selecciona Sin conservación de datos, NO se guardará ningún dato. Cuando utilice la API para descargar datos, estos no estarán disponibles.

    Crear un reproductor habilitado para Audience

    Los datos de interacción de vídeo solo se guardarán cuando utilice un reproductor de vídeo habilitado para el público. Para ver cómo crear un reproductor de Brightcove habilitado para Audience, consulte Crear un reproductor habilitado para Audience.

    Pasar un identificador de seguimiento

    El módulo Audience admite la capacidad de pasar un ID de seguimiento. Por ejemplo, el identificador de seguimiento se puede utilizar para realizar un seguimiento de un ID de campaña. El ID de seguimiento se puede agregar a la URL que muestra el reproductor habilitado para audiencia-o añadirse al código de inserción del reproductor.

    • Ejemplo: Añadido a la URL
      http://www.myurl.com/video.html?bctrackingid=12345
    • Ejemplo: Añadido al código de inserción del reproductor
    <video data-video-id="6033402539001"
        data-account="1486906377" Cover case Multicolor
        data-player="predeterminado»
        data-embed="predeterminado»
        id. de aplicación de datos
        class="video-js»
        data-bc-trackingid="12345"
        controles > < /vídeo >
<script src=» //players.brightcove.net/1486906377/default_default_default/index.min.js» > < /script >

    Identificar a los espectadores

    Cuando utilice una integración personalizada, usted será el responsable de identificar a los espectadores. El complemento plug-in Audience requiere dos datos fundamentales:

    Descripción Parámetro URL Parámetro Incrustado Valor
    Identificación de espectadores conocidos (necesario para el formulario de captación de clientes potenciales) bcknownuser data-bc-known-user 0 o 1
    GUID (necesario para el seguimiento de los eventos de visionado) bccustomguid data-bc-custom-guid string
     

    Estos valores pueden transmitirse con los parámetros de la dirección URL o a través del código de incrustación del reproductor. Para que haya un seguimiento de la audiencia, el identificador del usuario debe transmitirse en forma de parámetro de la dirección URL en la página en la que se está reproduciendo el vídeo. El parámetro de usuario conocido es necesario para los formularios de prospectos (si desea que el formulario de prospecto se muestre condicionalmente para espectadores desconocidos).

    Nota: Para que el reproductor/plugin lea los parámetros de URL, se debe usar el código de inserción avanzado (en la página). El código de inserción estándar (iframe) no funcionará ya que el complemento se ejecutará dentro de un iframe y, por lo tanto, no puede acceder a la URL de la página principal que tiene el parámetro.

    Visualización de datos de vídeo

    Existen varias opciones disponibles para ver datos de vídeo cuando se utiliza una integración personalizada:

    • Actividades del visor (eventos): esta opción se puede utilizar como parte del módulo Audience para ver datos de vídeo dentro del módulo Audience. Para obtener más información, consulte Visualización de eventos del Reproductor en el Módulo de audiencia.
    • Exportar datos: se proporciona como parte del módulo Audience, esta opción se puede utilizar para exportar la vista de vídeo y los datos de prospectos. Para obtener información, consulte Exportación de datos de audiencia desde el módulo de audiencia.
    • API de audiencia : la API de audiencia se puede utilizar para recuperar mediante programación datos de vista de vídeo y prospectos. Para obtener más información, consulte Visión general: API de audiencia.
