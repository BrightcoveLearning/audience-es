    Introducción al módulo Audience

    En este tema se proporciona una descripción general del módulo Audience que se utiliza para realizar un seguimiento de la interacción con el vídeo.
     

    El módulo Audiencia se utiliza para realizar un seguimiento de los datos de interacción de vídeo de vídeos que se visualizan dentro de un reproductor de Brightcove habilitado para audiencias. La visualización de datos se almacena en Video Cloud y, opcionalmente, se pueden sincronizar con plataformas populares de automatización de marketing como Eloqua, Marketo, HubSpot y Salesforce.

    Para acceder al módulo Audience, inicie sesión en Video Cloud y haga clic en Audiencia en el encabezado de navegación. La página principal de Audiencia muestra datos de interacción de vídeo que proporcionan información sobre cuánto tiempo se vieron los vídeos, qué vídeos se vieron y quién los vio. Con esta información podrá adaptar sus contenidos para atraer audiencias y convertir a sus espectadores en clientes.

    En la página principal se pueden realizar las siguientes operaciones:

    • Ver rendimiento de vídeo
    • Ver rendimiento de los clientes potenciales
    • Ver el estado del módulo Audience

    El módulo Audience proporciona la capacidad de:

    • Configurar conexiones de datos
    • Crear reproductores de audiencia
    • Crear y administrar formularios de prospectos
    • Publicar en correo electrónico
    • Ver eventos del jugador
    • Ver perfiles de visor
    • Ver actividad de sincronización
    • Configurar la configuración de audiencia
    • Exportar datos de audiencia
    • Exportación de clientes potenciales
    • Administrar datos del visor

    Ver el rendimiento de los vídeos

    La sección de rendimiento de los vídeos ofrece información de alto nivel sobre el comportamiento de los vídeos visualizados en reproductores habilitados para Audience.

    Nota: El sistema de análisis de Audience es independiente del sistema general de análisis de Video Cloud y los dos realizan un seguimiento diferente de los datos. Por lo tanto, verá diferencias en algunas métricas si compara las dos. Por ejemplo, el sistema de análisis de Video Cloud cuenta las vistas tan pronto como se inicia un vídeo, mientras que el sistema de análisis de audiencia no cuenta las vistas hasta 10 segundos en el vídeo. Por lo tanto, si 14 espectadores inician el vídeo, pero 6 de ellos lo detienen antes de que se hayan reproducido 10 segundos, los análisis de Video Cloud mostrarían 14 vistas, mientras que el módulo Audience mostraría 8.

    Comience seleccionando un periodo de tiempo para los gráficos. Están disponibles las siguientes opciones

    • 7 días
    • 30 días
    • 90 días

    Vídeos más vistos

    El gráfico Vídeos más vistos muestra los vídeos más vistos en reproductores habilitados para audición durante el período de tiempo seleccionado.

    Vídeos de mayor interés despertado

    El gráfico Vídeos más atractivos muestra los vídeos más atractivos en reproductores habilitados para el público durante el período de tiempo seleccionado. El índice de interés despertado es el promedio porcentual de todos los tiempos de visionado. Para obtener más información sobre cómo calculamos la interacción con el vídeo, vea el vídeo de formación Calculación de la puntuación de compromiso de vídeo.

    Tendencias de los espectadores

    El gráfico Tendencias del visor muestra vistas de vídeo a lo largo del tiempo durante el período de tiempo seleccionado. Se indican los visionados de espectadores tanto conocidos como anónimos. A medida que se desplaza el puntero sobre el gráfico, van apareciendo los datos subyacentes.

    Nota: Cuando se utiliza una conexión Marketo Munchkin, el gráfico Tendencias del visor solo mostrará los datos de Visores Desconocidos. Cuando se utiliza una conexión de Munchkin, los detalles del usuario no se recopilan. Para obtener información más detallada del usuario, se debe utilizar la conexión REST de Marketo. Para obtener información, consulte Integración de Video Cloud con Marketo Using the REST API.

    Visualización del rendimiento de clientes

    Para ver la información de prospectos, haga clic en la ficha Leads . Ten en cuenta que la pestaña Leads solo aparecerá si tu cuenta tiene formularios de clientes potenciales de audiencia creados. La sección de rendimiento de clientes potenciales proporciona un rendimiento de clientes potenciales de alto nivel para vídeos que se reprodujeron dentro de reproductores habilitados para público. Comience seleccionando un periodo de tiempo para los gráficos. Están disponibles las siguientes opciones

    • 7 días
    • 30 días
    • 90 días

    Se muestran los siguientes diagramas:

    • Embudo del público
    • Vídeos principales de conversión
    • Tabla de clientes potenciales

    Embudo del público

    El embudo de audiencia muestra las métricas de los vídeos en reproductores habilitados para audiencia con formularios de prospectos de audiencia para el período de tiempo seleccionado:

    • Impresiones de vídeo - El número de vídeos que se cargaron
    • Visualizaciones de vídeo - El número de veces que un vídeo comenzó a reproducirse, grabado cuando se inicia la transmisión (no incluye rebobinado o repetición); no es una medida de espectadores individuales
    • Vistas de formulario de prospecto : el número total de vistas de formulario de prospecto
    • Conversiones : el número total de formularios de prospecto enviados

    Vídeos principales de conversión

    La tabla Top Converting Videos muestra los vídeos con la tasa de conversión más alta. La tasa de conversión se calcula dividiendo el número de formularios de clientes potenciales enviados dividido por impresiones de vídeo.

    Tabla de clientes potenciales

    El diagrama prospectos muestra el número de formularios de prospectos enviados por día para el período de tiempo seleccionado. A medida que se desplaza el puntero sobre el gráfico, van apareciendo los datos subyacentes.

    Visualización del estado del módulo Audience

    La sección Estado de audiencia muestra el estado actual del módulo Audience.

    Configuración de conexiones de datos

    La opción Conexiones de datos es donde se configuran las integraciones de audiencia. La audiencia puede integrarse con plataformas de automatización de marketing populares (« integración de MAP») o los datos de visualización se pueden guardar localmente y luego exportarse o extraerse mediante la API de audiencia (integración «no Map»). En la siguiente tabla se describen las integraciones admitidas.

    Integraciones de MAP Integraciones sin mapa
    • Oráculo Eloqua
    • Marketo (API REST)
    • Marketo (Munchkin)
    • Salesforce
    • HubSpot (API REST)
    • HubSpot (API del lado del cliente)
    • Integraciones personalizadas
    • Inicio de sesión único
    • Seguimiento de audiencia

    Al publicar reproductores habilitados para audición, utilice el código de publicación Advanced Insert (en página). El código de publicación Standard Insert (iframe) no funcionará ya que el complemento Audience se ejecutará dentro de un iframe y, por lo tanto, no podrá acceder a ningún parámetro de URL de página principal y no obtener las cookies correctas.

    Nota: El número de conexiones que se pueden configurar depende de su suscripción a Video Cloud.

    Creación de reproductores de audiencia

    La opción Reproductores de audiencia permite crear reproductores de Brightcove habilitados para audiencias. Los datos de interacción de vídeo solo se grabarán cuando el contenido de vídeo se vea dentro de un reproductor de vídeo habilitado para el público. Para obtener información sobre cómo crear reproductores habilitados para audiencias, consulte Creación de un reproductor habilitado para audiencia-habilitado.

    Crear y administrar formularios de clientes potenciales

    La opción Formularios de prospectos permite crear formularios de prospectos para la captura de datos. Una vez creado el formulario de captación de clientes potenciales, se puede asociar a un reproductor habilitado para Audience. Para obtener información sobre la creación de formularios de prospectos, consulte Creación de un formulario de prospecto de audiencia. También es posible crear y utilizar formularios de clientes potenciales que se crean dentro de una plataforma de automatización de marketing de terceros.

    Publicar un vídeo en correo electrónico

    La opción Publicar en correo electrónico le permite seleccionar un reproductor, vídeo y, a continuación, generar el código de inserción HTML para enviar el vídeo por correo electrónico. Para obtener más información, consulte Publicación de un vídeo en correo electrónico mediante el módulo de audiencia.

    Nota: El módulo Media también proporciona la capacidad de publicar un vídeo a correo electrónico. Para obtener más información, consulte Publicación de un vídeo en correo electrónico.

    Ver eventos de reproductores

    La opción Todas las actividades recientes le permite ver los eventos de la vista del jugador que se han grabado durante las últimas 24 horas. Para obtener más información, consulte Visualización de eventos del Reproductor en el Módulo de audiencia.

    Ver actividad de vídeo

    La opción Vídeo (sin mapa) le permite ver la actividad de visualización de un vídeo. Para obtener más información, consulte Visualización de la actividad de vídeo.

    Visualización de perfiles de visor

    La opción Perfil del visor (no mapa) le permite ver los eventos de visualización del reproductor por espectador. Esta opción está habilitada cuando se utiliza SSO o integraciones personalizadas, o Seguimiento de audiencia. Para obtener más información, consulte Visualización de la información del perfil del visor.

    Visualización de la actividad de sincronización

    La opción Actividad de sincronización muestra los últimos 10 trabajos que se sincronizaron de Audiencia a una plataforma de automatización de marketing. La sincronización se ejecuta cada hora. Seleccione una conexión y, a continuación, haga clic en el botón Probar sincronización para ejecutar una sincronización. Esto resulta útil cuando se prueban nuevas integraciones para evitar tener que esperar para ver si los datos de eventos se están enviando a su plataforma de automatización de marketing. Test Sync se puede ejecutar una vez cada 5 minutos hasta 10 veces por hora.

    Nota: Cuando se utilizan integraciones personalizadas, no se crean trabajos de sincronización. Los trabajos de sincronización solo se crean cuando se utilizan integraciones con una plataforma de automatización de marketing.

    Configuración de la audiencia

    La opción Configuración le permite configurar algunos parámetros predeterminados para la cuenta. Para obtener más información, consulte Configuración de la audiencia.

    Exportar datos de audiencia

    La opción Exportar datos de audiencia se utiliza para exportar datos de eventos de vista y de formulario de prospecto. Esta opción sólo aparecerá para los administradores de cuentas. Para obtener información sobre cómo exportar datos de audiencia, consulte Exportación de datos de audiencia desde el módulo de audiencia.

    Exportación de clientes potenciales

    La opción Exportar clientes potenciales de la galería se utiliza para exportar datos de formularios de prospectos capturados a partir de experiencias de Galería. Esta opción sólo aparecerá para los administradores de cuentas. Para obtener más información sobre la exportación de datos de prospecto de Galería, consulte Exportación de datos de prospecto

    Administración de datos del visor

    La opción Administrar datos del visor permite buscar un usuario y, a continuación, quitar ese usuario o exportar los datos de ese usuario. Esta opción sólo aparecerá para los administradores de cuentas. Para obtener información sobre la administración de datos del visor, consulte Administración de datos del visor.
